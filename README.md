# Aplikasi Chatting

## Requierment
- jdk

### Input: -

Server harus berjalan agar input pengguna dapat diproses, setelah server berjalan pengguna dapat menggunakan bidang teks, tombol, dan area teks untuk memberikan input ke sistem.

- User dapat menggunakan nama berlabel bidang teks untuk memasukkan nama pengguna.
- User kemudian dapat menekan gabung untuk mengirim nama pengguna ke server untuk diproses.
- Jika nama tersebut diterima oleh server, maka pengguna dapat menggunakan area teks berlabel tulis untuk membuat pesan yang akan dikirim ke semua orang yang bergabung dengan ruang obrolan / server.
- Setelah obrolan dibuat, pengguna dapat menekan tombol kirim untuk mengirim pesan ke server untuk disiarkan ke semua orang yang bergabung di server obrolan.
- Pengguna dapat menekan keluar untuk mengakhiri koneksi dan menutup aplikasi.
- Tugas yang sama dapat dilakukan dengan menekan tombol silang di UI Windows.

#### client:

Client akan menampilkan pengguna dengan daftar pengguna aktif di server obrolan serta pesan yang telah dibagikan oleh pengguna serta pemberitahuan yang dikirim oleh server untuk memberi tahu pengguna tentang pengguna baru yang bergabung atau meninggalkan obrolan.

- Pesan berlabel tampilan daftar menampilkan semua pesan pengguna serta pemberitahuan yang dikirim oleh server tentang pengguna yang keluar dan bergabung dengan server obrolan.
- Tampilan daftar pengguna aktif berlabel menampilkan semua pengguna yang terhubung ke server tidak termasuk dirinya sendiri.
